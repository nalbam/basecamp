# zsh

## zsh for linux

```bash
yum -y install zsh

chsh -s /bin/zsh
```

## zsh for mac

```bash
brew install zsh

chsh -s /bin/zsh
```

## oh-my-zsh

```bash
sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

vi .zshrc

# ZSH_THEME="dpoggi"

source ~/.zshrc
```
