==== Yum List ====
<code bash>
yum list
yum grouplist
</code>

==== Install ====
<code bash>
yum -y groupinstall "Web Server"

yum -y groupinstall "PHP Support"

yum -y groupinstall "MySQL Database client"
yum -y groupinstall "MySQL Database server"
 
yum -y install phpMyAdmin
</code>

==== Update ====
<code bash>
yum -y update
</code>
